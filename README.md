# 3D 手势交互贪吃蛇 (3D Snake Game)

这是一个基于 Vue 3 + TypeScript + Vite + Three.js + MediaPipe 开发的实时 3D 手势交互贪吃蛇游戏。

## ✨ 主要功能

- **👋 手势控制**：通过摄像头捕捉手部动作，移动手掌即可控制贪吃蛇的移动方向。
- **🚀 捏合加速**：捏合拇指和食指，触发加速模式 (Speed Boost)。
- **🎨 个性化定制**：
  - 支持切换多种蛇身模型（方块、球体、四面体）。
  - 支持实时调整蛇身发光颜色。
- **📱 多端适配**：完美适配 PC 端和移动端（手机）屏幕，自动调整 UI 布局。
- **📷 智能摄像头**：在手机上自动调用前置摄像头。

## 🛠️ 技术栈

- **前端框架**: Vue 3 + TypeScript + Vite
- **3D 引擎**: Three.js
- **AI 视觉**: MediaPipe Hands

## 🚀 快速开始

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```

3. **开始游戏**：
   - 打开浏览器访问控制台显示的地址（如 `http://localhost:5173`）。
   - **允许浏览器访问摄像头权限**。
   - 举起一只手，对准摄像头即可开始控制。

## 🎮 操作指南

- **移动方向**：在摄像头视野内上下左右移动手掌，蛇头会跟随手掌的位置移动。
- **加速冲刺**：将拇指和食指捏合在一起。
- **全屏模式**：点击控制面板上的 "Toggle Fullscreen" 按钮进入沉浸式体验。

## ⚠️ 注意事项

- **光线环境**：请确保环境光线充足，以便摄像头能准确识别手势。
- **手机测试**：如果在手机上测试，请确保手机和电脑在同一局域网，并使用 Network IP 访问。部分浏览器可能要求 HTTPS 才能调用摄像头。
